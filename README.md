## Prerequisitos
- Sistema GNU/Linux (Probado en arch)
- OpenJDK 20+ 
- Maven 3.8+
## Instalacion
__Clona el repositorio:__
```
git clone https://github.com/Matez204/Open-Synthesia.git
```
__Abrir la carpeta del proyecto__
```
cd Open-Synthesia
```
__Ahora instalar las depencias de javaFX con mvn__
```
mvn clean install
```
__Y por ultimo ejecutar el programa con maven__
```
mvn javafx:run
```
